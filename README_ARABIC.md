# Multi-Lingual Emotion Words Mapping: Korean, English, and Arabic

</section>

## نظرة عامة
يُعد هذا المشروع امتداداً وتطويراً لمجموعة بيانات **EVOKE** (مفردات المشاعر في اللغتين الكورية والإنجليزية). تقدم مجموعة بيانات EVOKE الأصلية خريطة دقيقة وشاملة للترجمات المتعددة (Many-to-Many) لكلمات المشاعر بين الكورية والإنجليزية. من خلال دمج الطبقة اللغوية العربية باستخدام مسارات عمل الترجمة الآلية، يهدف هذا المستودع إلى توسيع النطاق اللغوي المشترك لدلالات المشاعر. هذا التوسع يخدم بشكل مباشر مجالات معالجة اللغات الطبيعية (NLP)، علم اللغة النفسي (Psycholinguistics)، واللغويات الحاسوبية، مما يسهل بناء أنظمة ذكاء اصطناعي قادرة على فهم الفروق الدقيقة للمشاعر عبر الثقافات المختلفة.

## الأهداف الرئيسية
1. **إثراء الذكاء الاصطناعي متعدد اللغات:** توفير قاعدة بيانات قوية لتدريب نماذج تحليل المشاعر (Sentiment Analysis) التي تتطلب فهماً دقيقاً للتقاطعات اللغوية بين لغات شرق آسيا، واللغات الغربية، واللغات السامية.
2. **الترجمة السياقية الدقيقة:** تجاوز الترجمة الحرفية من خلال توفير تعيينات متعددة للكلمة الواحدة، مما يعكس الطبيعة المعقدة للمشاعر الإنسانية وكيفية التعبير عنها في اللغة العربية مقارنة بالكورية والإنجليزية.
3. **أتمتة موثوقة للبيانات:** تقديم نصوص برمجية جاهزة للاستخدام للتعامل مع واجهات برمجة التطبيقات (APIs) الخاصة بالترجمة بطريقة تتجنب قيود الحظر (Rate Limits) وتضمن سلامة البيانات.

## المميزات التقنية
- **توسيع مجموعة EVOKE:** إضافة عمود الكلمات العربية ليتزامن مع البيانات الكورية والإنجليزية.
- **ترجمة آلية قوية:** استخدام مكتبة `deep-translator` لضمان استقرار الاتصال بخوادم الترجمة وتجنب الحاجة لمفاتيح API معقدة.
- **دعم النصوص من اليمين إلى اليسار (RTL):** معالجة وحفظ البيانات بترميز `utf-8-sig` لضمان قراءة الحروف العربية بشكل صحيح عند فتح الملفات في برامج الجداول الإلكترونية مثل Microsoft Excel.
- **إدارة الأخطاء والقيود:** تطبيق تأخيرات زمنية (Time delays) ذكية بين الطلبات للحفاظ على استمرارية جلب البيانات دون التعرض للحظر المؤقت من مزود خدمة الترجمة.

## التثبيت والإعداد

1. **استنساخ أو تنزيل المستودع**
   تأكد من وجود جميع ملفات المشروع، بما في ذلك مجموعة البيانات الأساسية (`Emotion_Words_Combined_Clean.csv`)، في مجلد العمل الخاص بك.

2. **إعداد البيئة الافتراضية (Virtual Environment)**
   يُنصح دائماً بعزل بيئة العمل البرمجية:

```bash
   python -m venv env
   # في أنظمة Windows:
   env\Scripts\activate
   # في أنظمة macOS/Linux:
   source env/bin/activate

```

3. **تثبيت المكتبات المطلوبة**
   
```bash
pip install pandas deep-translator

```



## طريقة الاستخدام

1. **التحقق من البيئة:** قم بتشغيل نص التحقق البرمجي للتأكد من تثبيت جميع الاعتماديات ووجود مجموعة البيانات في المكان الصحيح.

```bash
python verify_env.py

```


1. **تنفيذ عملية الترجمة:** قم بتشغيل النص البرمجي الرئيسي لمعالجة ملف CSV. سيتم طباعة تقدم العملية في وحدة التحكم (Console).

```bash
python main.py

```


*سيتم حفظ المخرجات النهائية في ملف جديد باسم `Emotion_Words_Combined_Arabic.csv`.*
